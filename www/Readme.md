Readme.md

El index.html esta relacionado js/beeter.js.
Al abrir el index encontramos las siguientes funcionalidades:
- Al cargar, lista los 6 ultimos stings añadidos.
- Podemos añadir un Sting
- Opcion de busqueda por usuario y contenido
- Editar un Sting (Abriendose una ventana modal)
- Eliminar un Sting
- Funcion de validate en caso de dejar cualquier campo vacio.

El usuario esta predefinido, por lo tanto crearemos y editaremos con el usuario
y password alicia:alicia.

La url de funcionamiento es la raiz del proyecto:
http://localhost:8080/beeter/index.html
